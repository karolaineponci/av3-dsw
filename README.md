## 🛠️ Como Rodar o Projeto

### 1. Banco de Dados

1. Abra o seu MySQL (via Workbench ou terminal).
2. Importe o arquivo `dump.sql` que está na raiz deste repositório para restaurar a estrutura das tabelas e os dados de teste.

### 2. Backend

1. Abra o terminal na pasta `/backend`.
2. Instale as dependências executando:

```bash
bun install
```

3. Inicie o servidor em modo de desenvolvimento com o comando:

```bash
bun dev
```

(O servidor rodará na porta 3000)

### 3. Frontend

1. Abra a pasta `/frontend`.
2. Execute o arquivo `login.html` utilizando a extensão Live Server do VS Code (ou qualquer servidor local de sua preferência) para rodar na porta 5500.
